# 语义检索技能

## 功能说明
- 支持两种任务模式：
  1. `similarity`: 计算两个文本的语义相似度得分（0-1之间，数值越高相似度越高）
  2. `retrieve`: 从多个候选文档中召回与查询文本语义最接近的`top_k`个文档，返回得分和对应文本

## 依赖
- Python 3.8+
- 预训练语义向量模型（已内置在核心脚本中）

## 使用流程
### 1. 调用方式
执行内置脚本完成任务：
```bash
python <skill_path>/scripts/retrieve.py --query "<查询文本>" --documents "<候选文档1>;<候选文档2>;..." --task <任务模式> --top_k <召回数量>
```

### 2. 参数说明
| 参数 | 说明 | 必填 |
|------|------|------|
| --query | 查询文本 | 是 |
| --documents | 候选文档集合，多个文档用英文分号`;`分隔 | 是 |
| --task | 任务模式：`similarity`或`retrieve` | 是 |
| --top_k | 召回数量，默认1个文档 | 否 |

### 3. 输出格式
#### similarity模式
直接返回相似度得分，保留4位小数，示例：
```
0.9141
```

#### retrieve模式
返回得分最高的文档和对应得分，示例：
```
得分：0.6883
句子：闲暇时刻约上三五好友踢一场野球，输赢早已不重要，场上的欢声笑语和赛后围坐在一起喝冰汽水的时光，是属于足球爱好者独有的快乐。
```

## 示例
### 示例1：计算两个文本相似度
输入：
```
query: "我很喜欢看电影和听音乐"
documents: "我经常看电影和听音乐"
task: similarity
```
输出：
```
0.9141
```

### 示例2：召回最相似文本
输入：
```
query: "周末的社区足球场上，不同年龄的爱好者们迎着夕阳奔跑，汗水落在草地上的痕迹是对这项运动最纯粹的热爱注脚。"
documents: "卡塔尔世界杯决赛上，梅西捧起大力神杯的瞬间，无数守候在屏幕前的球迷激动得热泪盈眶，共同见证了足坛新传奇的诞生。;每一次精准的传切配合、每一脚势大力沉的射门、每一次奋不顾身的扑救，都是足球这项运动令人心潮澎湃的魅力所在。;不少城市的中小学都开设了足球特色课程，越来越多的孩子从小接触这项运动，为中国足球的未来积蓄着新鲜力量。;闲暇时刻约上三五好友踢一场野球，输赢早已不重要，场上的欢声笑语和赛后围坐在一起喝冰汽水的时光，是属于足球爱好者独有的快乐。"
task: retrieve
```
输出：
```
得分：0.6883
句子：闲暇时刻约上三五好友踢一场野球，输赢早已不重要，场上的欢声笑语和赛后围坐在一起喝冰汽水的时光，是属于足球爱好者独有的快乐。
```
