# speech-analyze 音频分析技能
专业的语音识别与音频内容分析工具，支持语音转文本、背景音效识别、音频质量评估、语音理解等功能。

## 核心能力
1. **语音转写**：将音频中的人声内容精准转换为带标点的纯文本，无多余输出
2. **背景音效分析**：识别音频中的背景声音类型（如背景音乐、环境杂音、乐器伴奏等），分析其特点、音量特征、与人声的适配度
4. **语音理解**：根据音频中的语音内容，理解其含义、情感、意图等，输出结构化结果

## 使用流程
### 1. 语音转写场景
当用户需要将音频中的人声转换为文本时，执行以下命令：
```bash
.venv/bin/python [YOUR_SKILLS_DIR]/speech-analyze/scripts/speech.py --prompt "将这段语音中的话语转换成文本，只输出转录的文本，需要添加标点，但不要加入其它多余文字" --audio [音频文件绝对路径]
```
直接返回命令输出的转写结果即可，不需要添加额外说明。

### 2. 背景音效分析场景
当用户需要分析音频的背景声音时，执行以下命令：
```bash
.venv/bin/python [YOUR_SKILLS_DIR]/speech-analyze/scripts/speech.py --prompt "分析这段语音中的背景音效，详细说明背景声音类型、特点、音量特征，不要输出人声转写内容" --audio [音频文件绝对路径]
```
将返回的结果整理为结构化的分点说明输出。

### 3. 综合分析场景
当用户同时需要转写和背景分析时，依次执行上述两个命令，先输出转写结果，再输出背景分析结果。

## 输入要求
用户必须提供待处理音频文件的绝对路径，如果用户未提供，先询问音频路径再进行处理。

## 输出示例
### 语音转写输出示例
```
画，唐，王维。
远看山有色，近听水无声。
春去花还在，人来鸟不惊。
```

### 背景音效分析输出示例
```
### 背景音效分析结果：
1. **背景声音类型**：唯一的背景音效是适配古诗意境的古筝独奏古风纯音乐，是专门为诗词朗诵定制的铺垫伴奏。
2. **音效特点**：音色为古筝典型的清亮通透弹拨质感，单音带有自然的乐器腔体延音泛音，余韵舒缓悠远；开篇先弹出3个稀疏空灵的散板单音铺垫氛围，朗诵开始后转为平缓的分解和弦伴奏，无复杂突兀的演奏技巧，整体曲风清幽写意，与诗词意境高度契合。
3. **音量特征**：全程伴奏音量远低于前景人声，属于低存在感的铺垫层，音量稳定无剧烈波动，既不会干扰人声接收，又恰到好处地烘托了氛围。
```
